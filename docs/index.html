📦 Installation

macOS (Apple Silicon) curl -sSL https://github.com/zoe668668/cli-battery/releases/latest/download/cli-battery-darwin-arm64 -o cli-battery && chmod +x cli-battery Linux curl -sSL https://github.com/zoe668668/cli-battery/releases/latest/download/cli-battery-linux-amd64 -o cli-battery && chmod +x cli-battery

🚀 Quick Start

./cli-battery # Show battery status ./cli-battery --watch # Watch mode ./cli-battery --json # JSON output
